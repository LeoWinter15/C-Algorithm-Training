# C. Bracket Integer 题解

## 题目概述
给定一个正整数 \(A\)（无前导零，位数可达 \(2\times 10^5\)），要求找出**最大的**“括号整数” \(X\)，满足 \(X \le A\)。

**括号整数**定义：存在一个**平衡加权括号序列（BWBS）**，其第 \(i\) 个括号的权值恰好等于该整数的第 \(i\) 位数字。  
BWBS 的递归定义：
- 空序列是 BWBS；
- 若 \(P\) 是 BWBS，则 \((w\;P\;)_w\) 也是 BWBS（两端括号权值相同）；
- 若 \(P,Q\) 是 BWBS，则 \(P Q\) 也是 BWBS。

例如 `1000022122` 是括号整数，`1001` 也是，而 `1000` 不是（答案应为 `99`）。

---

## 解题思路

### 1. BWBS 的判定
将一个数字串视为括号序列，按顺序扫描：
- 维护一个**栈**；
- 若当前数字等于栈顶，则弹出（匹配一对括号）；
- 否则将当前数字压入栈。

最终栈为空当且仅当该数字串是 BWBS。

**关键性质**：扫描到位置 \(i\) 时，若栈大小为 \(k\)，剩余长度为 \(L\)，则**存在一种方式补全剩余数字使得最终栈为空**的充要条件是：
\[
k \le L \quad \text{且} \quad (L - k) \bmod 2 = 0
\]
因为栈中每个元素都需要一个匹配位置，剩余位置必须成对出现（新的括号对）。

### 2. 构造最大合法数
我们从高位到低位尝试构造答案，尽量保留原数 \(A\) 的数字。  
若在某一位无法继续（即压栈后不满足上述条件），则尝试在该位减小数字，并让后续位填充成**最大的可能 BWBS**。

由于数字只有 0~9，我们可以在每一位从大到小枚举候选数字，进行回溯搜索。  
- 最高位不能为 0（无前导零）。
- 当某一位已经小于原数对应位时，后续位可以取 0~9（不再受 \(A\) 限制）。

### 3. 特殊情况
- 若 \(A\) 的长度为奇数，则任何 BWBS 长度必须为偶数，因此答案最长为 \(|A|-1\)，此时直接输出 \(|A|-1\) 个 `9`（全 9 的偶数长度必然合法）。
- 若偶数长度搜索失败（即没有长度等于 \(|A|\) 的合法数 ≤ A），则输出 \(|A|-2\) 个 `9`（这是最大的偶数长度备选）。

---

## 算法步骤

1. **预处理**：读入字符串 \(s\)，令 \(n = |s|\)。
2. **若 \(n\) 为奇数**：输出 `string(n-1, '9')`。
3. **否则（\(n\) 为偶数）**：
   - 初始化答案字符串 `ans`，长度 \(n\)。
   - 初始化空栈 `st`。
   - 调用递归函数 `dfs(pos, less, st)`：
     - `pos`：当前处理位置（0-indexed）。
     - `less`：是否已经小于原数前缀。
     - 枚举当前位可填数字 \(x\)：
       - 上界：`less ? 9 : s[pos]-'0'`
       - 下界：`pos == 0 ? 1 : 0`（最高位不能为 0）
       - 模拟栈操作（匹配或压入）。
       - 检查剩余长度 `L = n - pos - 1` 是否满足 `k <= L && (L-k)%2==0`。
       - 若满足，则记录 `ans[pos]=x`，递归下一位。
       - 若后续递归成功，则返回 `true`；否则撤销栈操作，尝试下一个 \(x\)。
     - 若所有 \(x\) 均失败，返回 `false`。
   - 若 `dfs` 返回 `true`，输出 `ans`。
   - 否则输出 `string(n-2, '9')`。

---

## 复杂度分析
- 每位最多尝试 10 个数字，每次尝试栈操作 O(1)。
- 递归深度为 \(O(n)\)，总时间 \(O(10n)\)。
- 总输入位数不超过 \(2\times 10^5\)，因此时间复杂度为 \(O(10 \times 2\times 10^5) = O(2\times 10^6)\)，完全可行。

## C++17 代码

```cpp
#include <bits/stdc++.h>
using namespace std;

string s, ans;
int n;

bool dfs(int pos, bool less, vector<int>& st) {
    if (pos == n) return true;                       // 所有位已填完，成功
    int L = n - pos - 1;                             // 剩余位数
    int high = less ? 9 : (s[pos] - '0');            // 当前位最大可填数字
    int low = (pos == 0) ? 1 : 0;                    // 最高位不能为0
    for (int x = high; x >= low; --x) {
        bool pushed = false;
        // 模拟压栈/匹配
        if (!st.empty() && st.back() == x) {
            st.pop_back();
        } else {
            st.push_back(x);
            pushed = true;
        }
        // 可行性剪枝
        if ((int)st.size() <= L && (L - (int)st.size()) % 2 == 0) {
            ans[pos] = char('0' + x);
            if (dfs(pos + 1, less || (x < s[pos] - '0'), st)) {
                return true;
            }
        }
        // 撤销操作
        if (pushed) st.pop_back();
        else st.push_back(x);
    }
    return false;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int T;
    cin >> T;
    while (T--) {
        cin >> s;
        n = (int)s.size();

        // 奇数长度 → 答案长度必须为偶数
        if (n % 2 == 1) {
            cout << string(n - 1, '9') << '\n';
            continue;
        }

        ans.assign(n, ' ');
        vector<int> st;
        if (dfs(0, false, st)) {
            cout << ans << '\n';
        } else {
            // 偶数长度搜索失败 → 退一步，输出 n-2 个 9
            cout << string(n - 2, '9') << '\n';
        }
    }
    return 0;
}
```

## 注意事项
- 递归深度最大为 \(2\times 10^5\)，在 Codeforces 环境下通常可通过（栈空间默认足够）。若担心栈溢出，可改用迭代回溯。
- 最高位强制非零，避免输出带前导零的非法整数。
- 当长度为偶数但无法构造时，输出 `n-2` 个 `9` 是正确且最大的备选方案（因为 `n-2` 仍为偶数，且全 `9` 序列一定是 BWBS）。
